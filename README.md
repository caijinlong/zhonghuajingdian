# 中华经典古籍

中华古典核心经典收录，涵盖儒、道、兵、法、墨各家及经部、史部要籍，共计 17 部。每部书包含完整原文（或精选篇章）、**全文注音（拼音）** 及关键字词简注。

## 编写说明

- **全文注音**：每部书的原文均标注拼音，方便诵读学习。已完成的文章也将逐步补充注音。
- **篇幅较长的经典**（如《论语》二十篇、《道德经》八十一章等），采用**逐篇/逐章编写**的方式，避免单次编写内容过多导致中断或卡顿。
- 每篇文章包含：原文 + 注音 + 关键字词注释。
